Diagrams

<spring-boot-starter-webauthn> supports multiple flows

  • Authentication

  • Registration of a new user

  • Registration of a new device

  • Account recovery

Resources

Register new user

Registration start

A POST request is used to start the Add User ceremony

Request structure

POST /registration/start HTTP/1.1
Content-Type: application/json;charset=UTF-8
Accept: application/json
Content-Length: 72
Host: localhost:8080

{"username":"newjunit","registrationAddToken":null,"recoveryToken":null}
